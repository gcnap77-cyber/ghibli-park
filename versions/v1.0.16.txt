【變更檔案列表】
- style.css
- index.html

【詳細改動內容說明】
1. 繪製並美化互動地圖的園區邊界：
   - 於 style.css 新增 `.park-boundary-shape` 樣式，使用半透明的柔和森林綠 (#eef7ee) 作為填滿色，並搭配綠色虛線邊框 (#a5d6a7) 作為公園的區劃界線。
   - 於 index.html 中的 SVG 地圖背景之上、道路與互動區塊之下，置入符合愛・地球博紀念公園官方地圖走勢的貝茲曲線邊界路徑 (`<path class="park-boundary-shape" ... />`)，覆蓋整個主要景點與路網。

2. 區分並個別著色吉卜力公園 5 大園區：
   - 在 style.css 中，針對吉卜力大倉庫 (橘色 #d85c00)、青春之丘 (水藍色 #0077b6)、動動力森林 (森林綠 #277443)、魔法之里 (紫色 #7d409b) 和魔女之谷 (桃紅色 #d81b60) 分別定義專屬的實線邊框 (stroke) 與淡色填滿 (fill)。
   - 為各園區的 hover 狀態與 active 狀態定義個別的淡色填滿加深及相對應顏色的陰影濾鏡 (drop-shadow)，使滑鼠懸停與選取時有更精緻生動的微互動效果。
   - 優化當票種不支援時的遮罩狀態，確保 disabled 的 `.map-area-polygon` 使用 `!important` 強制套用灰色虛線外觀，不與彩色實線衝突。

3. 更新地圖圖例標示：
   - 將 index.html 地圖圖例文字更新，原「免費公共公園區」修正為「愛・地球博紀念公園範圍 (綠色虛線)」，原「吉卜力收費園區」修正為「吉卜力公園各園區 (實線彩色)」，與互動地圖上的實際繪圖表現完全對應。
   - 同時微調 CSS 中對應的圖例樣式 (`.legend-color.public` 與 `.legend-color.ghibli`)。

4. 升級資源快取更新戳記：
   - 將 index.html 中的 style.css 與 app.js 快取戳記升級至 `?v=16`，強制瀏覽器重載最新版樣式。
