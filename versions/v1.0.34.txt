版本號：v1.0.34
發布日期：2026-06-08
發布者：Antigravity
修正目標：新增日圓 (JPY) 兌台幣 (TWD) 即時匯率換算功能，支援自動 API 取得與備用離線匯率（0.197）Fallback 機制，更新快取戳記至 v=34。
===================================================================

【變更檔案列表】
- index.html
- style.css
- app.js

【詳細改動內容說明】
1. HTML 結構調整：
   - 價格欄位：大散步券（高級版/標準版）、里山散步券、地鐵 Linimo 聯外交通、APM 貓巴士等主要日圓價格處，新增 `data-jpy` 屬性與用於顯示台幣換算的 `<span class="twd-price">` 標籤。
   - 匯率資訊橫條：在購票區底部新增 `#currency-rate-info` 橫條，用以展示即時匯率更新狀態。

2. CSS 樣式新增：
   - 新增 `.twd-price` 樣式，使用 `em` 相對單位，自動配合父字體大小，並套用吉卜力森林綠配色（`#436d39` 和 `#3b5f32`）。
   - 設計 `#currency-rate-info` 微卡片樣式與背景毛玻璃效果。
   - 為貨幣換算圖示（`.currency-icon`）加入極具質感、12 秒循環一次的微旋轉動畫。

3. JS 互動邏輯：
   - 使用異步 fetch 呼叫免費免 Key 且由 CDN 驅動的公開匯率 API（`https://open.er-api.com/v6/latest/JPY`）。
   - 若成功獲取最新的 JPY/TWD 匯率，則自動為所有 `[data-jpy]` 換算並顯示格式化台幣價格（例如：`約 NT$ 1,438`）；若連線失敗，則主動切換至寫入 JavaScript 的備用匯率 `0.197`，確保網頁完全不跑版。
